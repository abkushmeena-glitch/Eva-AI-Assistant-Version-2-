# Eva-AI-Assistant (Version-2)
I have build a highly Advance AI Assistant named "Eva".

The backend part of our project develops 'Automation', 'Chatbot', 'Image generation', 'Real time search', 'Speech-to-Text', and 'Text-to-Speech'.

The frontend part focuses on Graphical User Interface (GUI) of our AI Assistant.

"Main.py" is the main file of our project which run all its features simultaneously.

The key features of our AI Assistant are:-

1. Real time search engine
2. Answer almost every query
3. It can write programming codes
4. Speech-to-Text and Text-to-Speech
5. It can generate images based on user prompts
6. Can open and close multiple apps and tabs simultaneously
7. It can write applications, letters and more using notepad by voice command
