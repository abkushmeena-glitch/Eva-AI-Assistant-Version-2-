# Eva-AI-Assistant (Version-2)
Eva is a highly advanced AI-powered assistant designed to automate tasks, answer queries, and interact with users through both voice and text.

It integrates powerful backend automation with a clean and interactive graphical user interface (GUI), creating a seamless AI experience.

# About The Project
Eva is built with a modular architecture where the backend manages intelligence and automation, while the frontend delivers a smooth and user-friendly interface.

The core of the project is "Main.py", which acts as the central controller and runs all features simultaneously, ensuring real-time performance and multi-tasking capabilities.

# Backend Capabilities
The backend system powers Eva with advanced AI functionalities such as:

- Intelligent Chatbot System
- Real-Time Web Search Engine
- AI-Based Image Generation
- Speech-to-Text (Voice Recognition)
- Text-to-Speech (AI Voice Response)
- Task Automation Engine
- Multi-process Handling

# Frontend
The frontend focuses on delivering a modern and interactive Graphical User Interface (GUI) that allows users to:

Interact using voice or text

View responses in real-time

Experience smooth AI-driven automation


# Key Features 

1. Performs real-time searches from the internet
2. Answers almost any general or technical query
3. Writes and explains programming code
4. Converts speech into text and responds with voice output
5. Generates images based on user prompts
6. Opens and closes multiple applications and browser tabs simultaneously
7. Writes applications, letters, and documents in Notepad using voice commands
8. Handles multiple tasks at the same time
